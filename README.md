## О программе

**Book Tracker** — десктопное приложение с графическим интерфейсом для ведения списка прочитанных книг.

### Основные функции:
- ➕ Добавление записи (название, автор, жанр, количество страниц)
- 📋 Просмотр всех книг в таблице
- 🔍 Фильтрация по жанру
- 🔍 Фильтрация по минимальному количеству страниц
- 💾 Сохранение в JSON-файл
- 📂 Загрузка из JSON-файла
- ❌ Удаление выбранной книги
- ✅ Проверка ввода (пустые поля, число страниц)
